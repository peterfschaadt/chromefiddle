About ChromeFiddle

ChromeFiddle was started as a personal project by Peter Schaadt to track the newest experimental features in Google's Chrome browser. As a web developer I am particularly interested in emerging web standards like NaCl, WebGL, and Web Intents. I hope that this site will encourage other users to enable and disable these experimental web components to create a browser environment that suits them.

Development Stack

ChromeFiddle was built using the loving combination of Python and the Django framework. Other tools used are PostgreSQL, SQLite3, fabric, and virtualenv. This site is Made in the USA in Sunnyvale, California and served by nginx and gunicorn in Chicago, Illinois. The source code repository is available on Github at https://github.com/peterfschaadt/chromefiddle. Feel free to contact me if you have any questions or suggestions!
