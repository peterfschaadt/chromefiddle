About ChromeFiddle

ChromeFiddle was started as a personal project by Peter Schaadt to track the newest experimental features in Google's Chrome browser. I am highly interested in emerging web standards and hope that this site will encourage other users to enable and disable these experimental web components to create a browser environment of their choosing.

Development Stack

ChromeFiddle was built using the loving combination of Python, Django, PostgreSQL, SQLite3, fabric, and virtualenv. This site is Made in the USA and hosted out of Chicago, Illinois. The source code repository is available on Github at https://github.com/peterfschaadt/chromefiddle. Feel free to contact me if you have any questions!
