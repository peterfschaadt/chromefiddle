About ChromeFiddle

ChromeFiddle was started as a personal project by Peter Schaadt.

Development Stack

ChromeFiddle was built using the loving combination of Python, Django, PostgreSQL, SQLite3, fabric, and virtualenv. This site is Made in the USA and hosted out of Chicago, Illinois. The source code repository is available on Github at https://github.com/peterfschaadt/chromefiddle. Feel free to contact me if you have any questions!
